# 名词短语中的形容词

这一类的**形容词**一般是出现在<b>限定词（像a、the、this、some、five、John's等词）与名词中间</b>。请观察以下的名词短语以及其中的形容词:    

| 限定词  |形容词   |名词   |   |
|:-:|---|---|---|
|three   |yellow   |roses   |（三朵黄玫瑰）   |
|a   |new   | camera  | （一架新相机）  |
|my   | best  | friend  | （我最好的朋友）  |
|   | dirty  | water  | （脏水）  |
|   |  pretty | women  |（漂亮女人）   |
